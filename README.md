# Sparkle Cleaning Solutions – Website Project  
**Web Development POE – Part 1, Part 2 & Part 3**

This repository contains a complete website developed for **Sparkle Cleaning Solutions**, a professional cleaning company offering home, office and deep-cleaning services in Pretoria.  
The website is created according to the POE requirements for **Part 1, Part 2, and Part 3**, including planning, structure, design, functionality, SEO and deployment.

# 3. PART 3 – Functionality, Interactivity & SEO

### ✔ JavaScript Added (`js/script.js`)
The site includes advanced interactive features:

Interactive Components
- FAQ accordion  
- Booking modal popup  
- Image lightbox on services page  
- Dynamic "special offers" loaded via JavaScript  
- Real-time service search/filter system  
- Map area dynamically populated with service locations  

 Form Validation (Client-side)
**Enquiry Form Validation**
- Full name validation  
- Email format validation  
- SA phone number validation (0XXXXXXXXX or +27XXXXXXXXX)  
- Service selection required  
- Additional details validation  

**Contact Form Validation**
- Name validation  
- Email validation  
- Message type required  
- Message length (min 10 characters)  

On successful validation:  
success message displayed  
On failure:  
error messages displayed
