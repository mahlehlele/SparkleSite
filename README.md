# Sparkle Cleaning Solutions – Website Project  
**Web Development POE – Part 1, Part 2 & Part 3**

This repository contains a complete website developed for **Sparkle Cleaning Solutions**, a professional cleaning company offering home, office and deep-cleaning services in Pretoria.  
The website is created according to the POE requirements for **Part 1, Part 2, and Part 3**, including planning, structure, design, functionality, SEO and deployment.

Part 3 focuses on enhancing website functionality and improving search engine optimisation (SEO). JavaScript was used to add interactivity and form validation, while SEO best practices were applied to improve visibility and performance.

Additional configuration files such as robots.txt and sitemap.xml we created them created to support search engine indexing.

Features Implemented
- JavaScript form validation
- Interactive user elements
- SEO-friendly page titles and meta descriptions
- Image optimisation with alt text
- Sitemap and robots.txt configuration

SEO Strategy
- Use of meaningful page titles and headings
- Clean and descriptive URLs
- Internal linking between pages
- Mobile-friendly design

Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub
